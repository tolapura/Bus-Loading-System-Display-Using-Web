BUS LOADING SYSTEM

Features

Features

Stambai is a full-blown bus loading system package which utilizes the power of mobile and web platforms and the RFID system in tracking the activities of the buses within the bus terminal, collecting relevant real-time data, generating violation and terminal fee collection reports, and ensuring that bus drivers can keep track of their stay within the loading base via an automatic alarm system. A real-time display system is also accessible to the public and the management.

Real-time Display

Automatically tracks the buses which are currently in a certain loading base, displays the bus info, calculates their duration of stay, and records violations if there are.

Collection Verification

Provides the management the data they can use to accurately verify and check the collected terminal fees daily, monthly, annually, or any given time duration.

Reports Generation

Generates the hard data used in calling out bus companies and security guards involved in violations. Reports generated can also be printed.

Data Collection

Contains an updatable and filterable database of all the buses in the terminal including the company, bus number, accommodation, seat cap, et. al.

Alarm Trigger

An alarm which turns green when a bus is loading, orange when 10 minutes is left, and red when a bus exceeds 30 minutes.

RFID Implementation

Each loading bay will be installed with an RFID reader which will detect newly arrived buses with rfid chip attached to it.
